1. Add items to your inventory config/database.

2. Add the required database data from SQL.sql file.

3. Check if the peuren_lib/shared/config.lua is setup correctly to your server and your dependencies.

4. Start the system in your server.cfg, this can be achieved by adding this line in your server.cfg: ensure peuren_burglary

5. Restart your server and enjoy our system!

6. If you have any questions or you are instrested on how the system works, you can read our documentation: https://peuren-development.gitbook.io/peuren/
